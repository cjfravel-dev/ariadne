  4.  def apply(name: String, schema: Option[StructType] = None, format: Option[String] = None, allowSchemaMismatch: Boolean = false, readOptions: Option[Map[String, String]] = None)(implicit spark: SparkSession): Index

    Primary factory method to create or reconnect to an Index instance.

    Primary factory method to create or reconnect to an Index instance.

    If metadata exists at the storage path, reconnects to the existing index and validates schema/format compatibility. If metadata does not exist, creates a new index with the provided schema and format.

    name

    Unique index name (must be a valid Hadoop path component).

    schema

    Optional Spark schema of the data files. Required for new indexes.

    format

    Optional data file format (e.g., "parquet", "csv"). Required for new indexes.

    allowSchemaMismatch

    When true and metadata exists, allows updating the stored schema. Validates that all existing index columns still exist in the new schema before applying the change.

    readOptions

    Optional map of read options for format-specific configuration (e.g., CSV delimiter).

    returns

    A fully initialized Index instance.

    Example: 

    1. // Reconnect to an existing index (no schema/format needed)
val index = Index("orders")

// Create a new index
val newIndex = Index("orders", Some(schema), Some("parquet"))

// Reconnect with schema evolution
val evolved = Index("orders", Some(newSchema), allowSchemaMismatch = true)
    Exceptions thrown

    FormatMismatchException if format differs from stored format

    IndexNotFoundInNewSchemaException if allowSchemaMismatch is true but an indexed column is missing

    MissingFormatException if creating a new index without a format

    SchemaMismatchException if schema differs and allowSchemaMismatch is false

    SchemaNotProvidedException if creating a new index without a schema
