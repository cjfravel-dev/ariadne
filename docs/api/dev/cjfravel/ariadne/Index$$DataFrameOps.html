DataFrameOps 

implicit class DataFrameOps extends AnyRef

Implicit enrichment enabling df.join(index, columns, joinType) syntax.

This provides the reverse join direction compared to Index.join: the driving DataFrame is on the left and the index-located data is on the right.

Usage:

import dev.cjfravel.ariadne.Index.DataFrameOps
val result = myDataFrame.join(index, Seq("user_id"), "inner")
