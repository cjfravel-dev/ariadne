AriadneScanBuilder 

class AriadneScanBuilder extends ScanBuilder with SupportsPushDownFilters

Builds a scan for reading Ariadne index source data with filter pushdown.

Implements SupportsPushDownFilters to convert WHERE clause filters on indexed columns into dev.cjfravel.ariadne.IndexQueryOperations.locateFiles calls, pruning the source file list before reading.

Thread safety: Instances are created per scan by AriadneTable.newScanBuilder and are not shared across threads.

See also

AriadneV1Scan for the scan implementation
