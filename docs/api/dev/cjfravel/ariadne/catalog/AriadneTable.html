AriadneTable 

case class AriadneTable(indexName: String, sourceSchema: StructType, metadata: IndexMetadata) extends Table with SupportsRead with Product with Serializable

Spark V2 table representing an Ariadne index's source data.

When queried, this table reads the original data files that were indexed (not the index metadata tables). The schema is the source data schema stored in the index's metadata.json.

This table supports batch reads only and is read-only. Write operations are not supported — index updates are performed via dev.cjfravel.ariadne.Index.update.

When used in a JOIN, the AriadneJoinRule optimizer rule rewrites the plan to use Ariadne's optimized file-pruning join path.

Thread safety: This is an immutable case class. Instances are safe for concurrent read access from multiple threads.

indexName

the name of the Ariadne index

sourceSchema

the schema of the original source data files

metadata

the index metadata containing format, indexes, and read options
