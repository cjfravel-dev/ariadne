## Reporting a Vulnerability

**Please do not open a public issue for security reports.** Instead:

- **GitHub:** [Report a security advisory](https://github.com/cjfravel-dev/ariadne/security/advisories/new) (preferred)
- **Email:** [ariadne-support@cjfravel.dev](mailto:ariadne-support@cjfravel.dev) with subject `[SECURITY] <brief description>`

See the [full security policy](https://cjfravel-dev.github.io/ariadne/contributors/security.html) for response timelines, supported versions, scope, and disclosure policy.
